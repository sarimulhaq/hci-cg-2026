Part B: True / False

  • Every link in a <ul> must be wrapped in <li>. True
  • A page may have multiple <h1> headings to enlarge text. False
  • aria-current="page" should only be on the active link. True
  • A form is accessible without <label> elements. False
  • KLM predicts execution time for expert, error-free tasks. True

Part C: Short Answer

Semantic vs Generic Elements: Semantic elements (like <nav>, <header>) carry meaning and help accessibility. Generic elements (like <div>) only provide structure.

Why KLM ignores high-level planning: KLM predicts only low-level action times (keystrokes, pointing, mental prep). It does not include interpretation, learning, or decision-making.
